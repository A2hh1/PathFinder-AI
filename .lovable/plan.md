

# PathFinder AI – Smart Career Navigation Platform

## Overview
A student-focused career guidance platform that analyzes academic transcripts, profiles, and achievements to recommend personalized career paths with AI-powered insights, skill-gap analysis, CV generation, and market opportunities. Built with React + Supabase + Lovable AI Gateway.

## Brand & Design
- **Primary:** #1E3A8A (deep blue) | **Secondary:** #14B8A6 (teal)
- Clean, modern, professional startup aesthetic
- Bilingual: English (default) + Arabic with full RTL support
- Responsive across desktop and mobile

---

- Hero section with value proposition and CTA

## Phase 2: Onboarding & Profile

### Multi-Step Onboarding Form
Collect in organized steps:
1. **Personal Info:** Full name, email, phone, profile image upload
2. **Academic Info:** University, college, major, academic year, GPA
3. **Skills & Interests:** Skills (tags), interests (tags), languages, career goals, preferred industries
4. **Transcript Upload:** PDF/image upload (mandatory), progress indicator

All data saved to Supabase (Firestore equivalent). Files stored in Supabase Storage.

### Profile Page
- View/edit all onboarding fields
- Upload/replace certificates, project files, profile image
- "Re-run AI Analysis" button after major changes
- Upload progress indicators, file validation

---

## Phase 3: AI Transcript Extraction

### Transcript Processing Flow
1. Upload PDF/image to Supabase Storage
2. Send to Lovable AI Gateway (Gemini) for extraction via edge function
3. Extract courses, grades, credit hours into structured data
4. Return confidence score
5. **Review screen:** Student sees extracted data in editable table
6. Student can manually correct entries before confirming
7. Confirmed data saved to Supabase

### Error Handling
- Retry on extraction failure
- Manual entry fallback if AI can't parse
- File type/size validation before upload

---

## Phase 4: Career Path Recommendation

### AI Analysis (Edge Function)
Weighted model sent to Gemini:
- Courses & Grades: 40%
- Projects: 25%
- Certifications: 20%
- Skills: 10%
- Interests: 5%
- Academic year relevance weighting

### Output
- 3–5 career paths with match percentage
- Clear explanation for each recommendation
- **Selection screen:** Student picks one path → saved to Supabase
- Selected path drives all subsequent features

## Database Schema (Supabase)

- **profiles** – name, email, phone, university, college, major, year, GPA, skills, interests, languages, career_goals, preferred_industries, profile_image_url
- **transcripts** – user_id, raw_file_url, extracted_data (JSON), confidence_score, confirmed
- **career_paths** – user_id, paths (JSON array), selected_path, analysis_date
- **skill_gaps** – user_id, missing_skills, roadmap, suggested_certs
- **readiness_scores** – user_id, total_score, breakdown (JSON)
- **uploads** – user_id, file_type, file_url, metadata, created_at
- **cv_data** – user_id, generated_cv (JSON), last_generated
- **chat_messages** – user_id, role, content, created_at
- **opportunities** – user_id, type (internship/job), data (JSON)

All tables with RLS policies scoped to authenticated user.

---

## Edge Functions
1. **extract-transcript** – Sends transcript to AI for parsing
2. **analyze-career** – Runs weighted career analysis
3. **generate-cv** – Builds CV content from profile data
4. **career-chat** – Streaming chatbot with student context
5. **generate-opportunities** – Creates personalized job/internship matches
6. **skill-gap-analysis** – Identifies gaps and generates roadmap
